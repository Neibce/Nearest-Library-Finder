# Nearest-Library-Finder

[한국어](./README.md)

A web service that searches for books by title, displays libraries where the selected book is available on a map, and shows the nearest library from your current location.

Developed in Q2 2021 (Term project for Internet Web Basics course)

## Features

- Search books by title
- Mark available libraries on the map
- Find the nearest library from current location
- Limited to Busan area (can be expanded nationwide by modifying the source code)

## Tech Stack

- Node.js (Express)
- KolisNet API (National Library of Korea)
- Kakao Map API

## How to Run

```bash
git clone https://github.com/Neibce/Nearest-Library-Finder.git
cd Nearest-Library-Finder
npm install
node index.js
```
